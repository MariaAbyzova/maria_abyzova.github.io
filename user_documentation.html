What is stored in Unicheck Institutional Library?

Unicheck Institutional Library stores:

  1. Files uploaded via Unicheck Standalone version by
    • Administrator
    • Instructors
    • Students
  2. Files uploaded via Unicheck Integrated version in your LMS, assessment app, another platform
  3. Uploaded databases

Unicheck Institutional Library content view:

Files uploaded via Unicheck Standalone

If you use Standalone version of Unicheck, all files and texts uploaded for similarity check via Unicheck Administrator account, Instructors and Students Standalone subaccounts, are added to Unicheck Library.

Files uploaded via Unicheck Integration

If under this account Administrator uses Integrated to LMS Unicheck version, all files and texts uploaded for similarity check by Students via LMS accounts, are added to Unicheck Library.

Uploaded databases

Unicheck Library might also contain your institutional database if you or Unicheck Team uploaded it on your request.

Note! Files stored in Unicheck Institutional Library are used to make similarity check of new submissions against your Library and detect matches. In an account of any level Similarity check against Library is always conducted against all your Unicheck Institutional Library content.
